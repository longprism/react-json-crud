# react-json-crud

Tutorial instal : 
1. Pastikan komputer terinstal NodeJS
2. clone repository
3. buka terminal
4. `cd react-json-crud`
5. `npm install`
6. `npm run dev`
7. buka terminal baru: ketikkan `npx json-server --watch  src/assets/data.json --port 8000`
8. buka website di browser

Terimakasih 🙏🙏
